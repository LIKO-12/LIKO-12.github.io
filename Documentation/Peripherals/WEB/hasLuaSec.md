# WEB.hasLuaSec
---

Checks if it has Luasec.

---

* **Available since:** _WEB:_ v1.0.0, _LIKO-12_: v0.6.0
* **Last updated in:** _WEB:_ v1.0.0, _LIKO-12_: v0.6.0

---

```lua
local hasLuaSec = WEB.hasLuaSec()
```

---
### Returns
---

* **hasLuaSec (boolean):** True if it has LuaSec, false otherwise.
