# GPU.screenWidth
---

Returns the width of the screen.

---

* **Available since:** _GPU:_ v1.0.0, _LIKO-12_: v0.6.0
* **Last updated in:** _GPU:_ v1.0.0, _LIKO-12_: v0.6.0

---

```lua
local w = GPU.screenWidth()
```

---
### Returns
---

* **w (number):** The width of the screen (By default - 192)
