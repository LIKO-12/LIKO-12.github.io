# GPU.screenHeight
---

Returns the height of the screen.

---

* **Available since:** _GPU:_ v1.0.0, _LIKO-12_: v0.6.0
* **Last updated in:** _GPU:_ v1.0.0, _LIKO-12_: v0.6.0

---

```lua
local h = GPU.screenHeight()
```

---
### Returns
---

* **h (number):** The height of the screen (By default - 128)
